

## About Laravel-Vue3

<p>vue3 and Laravel 11</p>
<p>The front-end uses vue3 and is integrated into Laravel 11</p>

## Running the tests

<p>Execute the following instructions to install</p>

laravel
Set APP_ROOT in .env
```
composer install
php artisan migrate
php artisan serve 
``` 
npm
``` 
npm install
npm run dev
npm run build
```
